批内技术分析命名空间

用于批内或截面计算，常和 over("datetime")、batch("datetime") 组合。

7 operatorsnamespace: batch_ta

算子列表

batch_ta.head

行算子，保留当前输入块前 size 行，其余位置返回 null。

batch_ta.lag

行算子，返回当前输入块向后看 size 行前的值。

batch_ta.lead

行算子，返回当前输入块向前看 size 行后的值。

batch_ta.rank

行算子，对当前输入块做批内排名（通常与 batch(...) 一起使用）。

batch_ta.slice

行算子，保留当前输入块 [offset, offset + length) 范围，其余位置返回 null。

batch_ta.sort

行算子，对当前输入块做批内排序，返回排序后的同长度结果。

batch_ta.zscore

行算子，对当前输入块做标准分数（z-score）。
