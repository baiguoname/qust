使用场景

用于策略信号、持仓状态、出入场和交易行号生成。

当你需要 stra.t02_track_entry_price 这种语义时，优先使用这个算子，而不是在 Python 里提前把数据取出来手写循环；这样才能保留流式状态、参数联动、Wasm 路径和 Monitor 重算能力。

为什么这样设计

策略和回测算子被设计成表达式，是为了让信号、持仓、成交、收益可以和 over/rolling/monitor 组合。这比在 Python for 循环里维护状态更容易复用，也能在分组和流式数据里保持一致语义。

实际收益是：策略信号、交易轮次、持仓收益和交易明细可以直接接到图表和 DataPool，方便定位每一笔交易。

注意事项

- 输入列顺序错误：这个算子按固定位置读取 3 列，不按列名匹配
- `open_sig` 长时间连续为 `True`：会在每行重置入场价，不等价于“仅开仓时打点”
- `open_sig=True` 但 `entry_now=null`：状态会被写成空值，后续依赖入场价的逻辑会变空
- 忘记接 `expanding()`：得到的不是你期望的完整时序列结果

**行算子**，跟踪“当前有效持仓的入场价”状态（用于后续止盈止损/回撤跟踪）📌

作用 ✅
----
这个算子把“开仓事件”对应的价格缓存为状态，在未平仓期间持续输出该入场价；
一旦平仓事件发生，状态清空并输出空值。

输入 📥
----
输入是 `col(open_sig, exit_sig, entry_now)`，顺序固定为 3 列：

1. `open_sig`: `bool | None`
2. `exit_sig`: `bool | None`
3. `entry_now`: `f64 | None`

说明：
- `open_sig` / `exit_sig` 的空值按 `False` 处理
- `entry_now` 通常是“本 bar 用于入场的参考价格”（如 `open_1`、`close_1`、滑点后价）

状态逻辑 🔄
--------
对每一行按如下顺序更新（先平后开）：

1. 若 `exit_sig=True`：先清空内部 `entry_price` 状态
2. 若 `open_sig=True`：再将 `entry_price = entry_now`
3. 输出当前 `entry_price`

因此在同一行同时出现 `exit_sig=True` 且 `open_sig=True` 时，
最终输出是“本行新的 `entry_now`”（即先平旧仓，再开新仓）。

返回 📤
----
返回 1 列：`entry_price`
- 类型：`f64`（可空）
- 语义：
  - 持仓期间：输出最近一次开仓记录的入场价
  - 非持仓期间：输出 `null`

上下文与计算语义 🧠
--------------
- 这是**行算子**
- 行算子若不接上下文算子（如 `expanding` / `rolling_*` / `group_by`），
  不具备“整列状态序列”语义
- 典型用法是：
  `col(...).stra.t02_track_entry_price().expanding()`

典型用途 🎯
--------
- 作为“开仓基准价”输入到其他状态算子（如 `t02_track_best_since_entry`）
- 构造 `pnl_from_entry = close / entry_price - 1`
- 和 `count_last` / `rolling_dynamic` 组合做开仓后区间统计

示例 💡
----
1. 基础示例（输出整列入场价轨迹）：

>>> data = pl.DataFrame({
...     "open_sig":  [False, True,  False, False, False, True,  False],
...     "exit_sig":  [False, False, False, True,  False, False, False],
...     "entry_now": [None,  101.0, 102.0, 103.0, 104.0, 110.0, 111.0],
... })
>>> qs.with_cols(
...     col("open_sig", "exit_sig", "entry_now")
...         .stra
...         .t02_track_entry_price()
...         .expanding()
...         .alias("entry_price")
... ).calc_data(data)

2. 同 bar 平仓并反手开仓（先平后开）：

>>> data = pl.DataFrame({
...     "open_sig":  [True],
...     "exit_sig":  [True],
...     "entry_now": [205.5],
... })
>>> # 输出 entry_price = 205.5
>>> qs.with_cols(
...     col("open_sig", "exit_sig", "entry_now")
...         .stra
...         .t02_track_entry_price()
...         .expanding()
...         .alias("entry_price")
... ).calc_data(data)

--------------

- `open_sig` 长时间连续为 `True`：会在每行重置入场价，不等价于“仅开仓时打点”
