kline.rl1m

kline.rl1m 是 K 线命名空间 下的算子。

namespace: kline returns: Expr

用法

Python
col("datetime", "open", "high", "low", "close", "volume").kline.rl1m

如果这个算子返回 Expr，通常可以继续链式调用 .alias(...)、.over(...)、.rolling(...) 或放入 qs.select(...)。

参数

无显式参数；输入来自当前表达式、绑定对象或当前命名空间。

输入和输出

项目说明
输入表达式当前表达式，例如 col("x")；具体列宽由底层算子校验。
返回类型Expr

完整例子 / Python
import polars as pl
import qust as qs
from qust import col

print("输入表达式:")
print('col("datetime", "open", "high", "low", "close", "volume")')
print("输出表达式:")
print('col("datetime", "open", "high", "low", "close", "volume").kline.rl1m')

data = pl.DataFrame(
    {
        "datetime": ["2026-01-01 09:30:00", "2026-01-01 09:31:00", "2026-01-01 09:32:00"],
        "open": [10.0, 10.2, 10.1],
        "high": [10.4, 10.5, 10.3],
        "low": [9.9, 10.0, 10.0],
        "close": [10.2, 10.1, 10.3],
        "volume": [1000, 1200, 900],
        "t": ["2026-01-01 09:30:00", "2026-01-01 09:31:00", "2026-01-01 09:32:00"],
        "c": [10.2, 10.1, 10.3],
        "v": [1000, 1200, 900],
        "bid1": [9.9, 10.0, 10.1],
        "ask1": [10.1, 10.2, 10.3],
        "bid1_v": [100, 120, 110],
    }
).with_columns(pl.col("datetime").str.to_datetime(), pl.col("t").str.to_datetime())

print("输入 DataFrame:")
print(data)

expr = col("datetime", "open", "high", "low", "close", "volume").kline.rl1m
df = qs.select(expr)
out = df.calc_data(data)

print("输出 DataFrame:")
print(out)

使用场景

用于 K 线、盘口和周期切片相关预处理。

当你需要 kline.rl1m 这种语义时，优先使用这个算子，而不是在 Python 里提前把数据取出来手写循环；这样才能保留流式状态、参数联动、Wasm 路径和 Monitor 重算能力。

为什么这样设计

Qust 把这一类能力做成表达式节点，是为了把“用户想算什么”保留下来。只要语义还在表达式树里，后续就可以统一处理分组上下文、窗口状态、参数联动、Wasm 执行和 Monitor 可视化。

实际收益是：代码更短，但语义更完整；后续计算、画图、调参、保存和回调都能复用同一棵表达式树。

注意事项

- 参数类型与预期不一致会导致运行时报错或返回空值。
- 在上下文算子链中使用时，请确认调用顺序与分组/窗口语义一致。
- 若上游列名、列顺序或数据类型变化，结果可能与示例不一致。
